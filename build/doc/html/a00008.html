HANDLE setupSerial ( const std::string & portName)

Sets up a serial communication connection.

Configures the serial port with standard parameters (9600 baud rate, 8 data bits, no parity, 1 stop bit) and returns a handle to the connection. If an error occurs, an error message is displayed, and INVALID_HANDLE_VALUE is returned.

Parameters
